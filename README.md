# sevillemitchell.github.io

Static site presenting a DevSecOps production delivery pipeline from developer commits through the production platform, including an interactive weighted metric fusion calculator for release readiness scoring.

## Local run

Open `index.html` directly in a browser, or serve the directory with any static file server.


## Release readiness score

The page includes a browser-based calculator that mirrors weighted Min-Max metric fusion for accuracy, error rate, and latency inputs.
